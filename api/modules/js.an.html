AnimTimerEvents

AnimTimerEvents: TimerEvents | "looping" | "looped"

The looped event is fired when an loop of an Animation ends, and another one begins. This event does not occur at the same time as the finished event, and therefore does not occur for animations with an animation-loop-count of one.

AnimTimerInit

AnimTimerInit: { delay?: number; duration?: number; loop?: boolean | number }

Type declaration

  • Optional delay?: number

    Delays (ms) the start of this animation.

  • Optional duration?: number

    Max duration time(ms) of this animation.

  • Optional loop?: boolean | number

    Defines max repeat number of this animation.
    The default value is 1.

    True means is Infinity.
    False means is 0.
