TWEEN ANIMATION

Init Parameters

Init a tween animation.

name type default info
keys JsonObject
<string|number|[number,number]>		 Targets's properties or attributes or styles
duration number 1000(ms)
delay number 0(ms)
endDelay number 0(ms)
direction 'forward'|'backword'|'alternate' 'forward'
loop boolean|number 1
autoreset boolean true
round number 3
easing function Easings.LINEAR

CODE EXAMPLE

new TweenAnim({
    keys: {
        translateX: 250
    },
    duration: 1000,
    delay: 500,
    endDelay: 500,
    direction: 'alternate',
    loop: true,
    autoreset: true,
    round: 0,
    easing: 'BACK_IN_OUT'
}).targets('.el');
