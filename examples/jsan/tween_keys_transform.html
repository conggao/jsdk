TWEEN ANIMATION

Keys: CSS Properties

Animate CSS transforms properties at same time.

Valid properties Default unit
'translateX''px'
'translateY''px'
'translateZ''px'
'rotate''deg'
'rotateX''deg'
'rotateY''deg'
'rotateZ''deg'
'scale'—
'scaleX'—
'scaleY'—
'scaleZ'—
'skew''deg'
'skewX''deg'
'skewY''deg'
'perspective''px'

CODE EXAMPLE

new TweenAnim({
    keys: {
        translateX: 250,
        scale: 2,
        rotate: '1turn'
    },
    easing: 'BACK_IN_OUT'
}).targets('.el');
