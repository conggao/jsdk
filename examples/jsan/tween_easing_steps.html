TWEEN ANIMATION

STEPS EASING Function

Defines the number of jumps an animation takes to arrive at its end value. 

easing: 'STEPS(numberOfSteps)'
PARAMETER DEFAULT MIN MAX
Number of steps 10 1 Infinity

CODE EXAMPLE

new TweenAnim({
    keys: {
        translateX: 250
    },
    loop: 3,
    duration: 5000,
    easing: 'STEPS(5)'
}).targets($L('.el'));
